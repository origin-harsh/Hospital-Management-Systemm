# Hospital Management System

A full-stack Hospital Management System to manage patients, doctors,
appointments, and hospital operations efficiently.

## Features
- Patient registration and management
- Doctor and staff management
- Appointment booking
- Secure authentication
- Role-based access

## Tech Stack
### Frontend
- React
- HTML, CSS, TypeScript

### Backend
- Spring Boot
- Java
- Maven

### Database
- MySQL

## Project Structure
Hospital-Management-System/
├── FrontEnd/hms/
├── BackEnd/
└── README.md

## How to Run the Project

### Backend
1. Open BackEnd in IDE
2. Run `HmsApplication.java`
3. Server runs on port 8080

### Frontend
1. Open FrontEnd folder
2. Run `npm install`
3. Run `npm start`
4. App runs on port 3000
